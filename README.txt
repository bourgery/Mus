Ce projet est un jeu de mus en C et SDL. (http://www.federation-mus.fr/regles1.html)

Ce projet possède deux versions la version1 pour jouer à 4 joueurs et la version2 pour jouer tout seul contre une IA. 
Pour lancer le jeu aller dans le dossier souhaité puis lancer la commande make puis make play
Pour quitter le jeu il faut appuyer sur “echap” lorsqu’on a une action a effectué
